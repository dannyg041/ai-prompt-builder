# AI Prompt Builder

## Overview
AI Prompt Builder is a simple web application that helps users create clear, structured prompts for large language models. It guides users through key prompt components such as role, goal, audience, tone, and constraints to improve AI output quality.

## Problem
Many people struggle to get useful results from AI tools because their prompts are vague, incomplete, or poorly structured.

## Solution
This app provides a lightweight interface that helps users think through the essential elements of a good prompt and generates a reusable, copy-ready prompt instantly.

## Features
- Role-based prompt framing
- Goal and audience definition
- Tone and output selection
- Constraint specification
- Copy-to-clipboard functionality
- Clean, minimal UI

## Tech Stack
- HTML
- CSS
- JavaScript

## How It Works
1. The user fills out the prompt fields
2. The app dynamically builds a structured prompt
3. The generated prompt can be copied and used in any AI tool
